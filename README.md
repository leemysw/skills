# Skills

个人 Skills 合集。

## Skills 列表

| Skill | 描述 |
|-------|------|
| [resume-coach](./resume-coach) | AI 简历评估与优化工具，基于 STAR/BAR 框架从猎头视角评分、分析、改写简历，支持多 JD 适配分析 |

## 如何使用

在 Claude Code 中告诉 agent：

> 帮我安装 https://github.com/leemysw/skills 的 skills

Agent 会自动拉取并安装到项目的 `.claude/skills/` 目录下。安装完成后通过触发词或直接提及 skill 名称来调用。
